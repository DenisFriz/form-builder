# Form Builder

This is a monorepo project containing a **frontend** and **backend** for a form builder application.

---

## Getting Started

Follow these steps to set up and run the project:

1. **Clone the repository:**

```bash
git clone https://github.com/DenisFriz/form-builder.git
```

## Navigate to the project folder:

```bash
cd form-builder
```

## Install dependencies for all workspaces:

```bash
npm install
```

## Start both frontend and backend concurrently:

```bash
npm run dev
```

## To run frontend tests, use the following command:

```bash
npm run test:frontend
```

## To run backend tests, use the following command:

```bash
npm run test:backend
```
